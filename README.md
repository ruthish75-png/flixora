# Flixora

Flixora is a responsive OTT-style movie browsing web app built with plain HTML, CSS, and JavaScript.

## Live Demo

- https://flixora-ruthi.vercel.app

## Highlights

- Responsive movie card layout for mobile, tablet, and desktop
- Hero showcase and category-based browsing rows
- Watch screen with related titles and detailed metadata
- Profile, settings, and support pages
- LocalStorage-based state for preferences and session-like behavior

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure

- `index.html` - Home and browsing experience
- `style.css` - Global styles and responsive breakpoints
- `script.js` - Main app interactions and rendering
- `watch.html`, `watch.js`, `watch.css` - Watch page experience
- `profiles.html`, `settings.html`, `support.html`, `login.html` - Supporting flows

## Local Run

1. Clone this repository.
2. Open `index.html` in your browser.
